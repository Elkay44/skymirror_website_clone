## Program Phases

### Phase 1: Foundation (Months 1-2)
- **Focus:** Onboarding, core skills, team integration
- **Learning:** 40% time
- **Deliverables:** 3-4 production features
- **Evaluation:** Month 2 review

### Phase 2: Mastery (Months 3-4)
- **Focus:** Independent ownership, advanced skills
- **Learning:** 25% time
- **Peer Mentoring:** 5% time
- **Deliverables:** 4-5 production features
- **Evaluation:** Month 4 mid-program review

### Phase 3: Leadership (Months 5-6)
- **Focus:** Module ownership, teaching, mentoring
- **Learning:** 15% time
- **Teaching:** 20% time
- **Deliverables:** 3-4 features + 5+ teaching sessions
- **Evaluation:** Month 6 final evaluation

### Extension Phase (Months 7-12)
- **Focus:** Senior intern/junior engineer hybrid
- **Teaching:** 30% time
- **Leadership:** Project leadership, mentor new cohorts
- **Deliverables:** 6-8 features + 10-20 teaching sessions/month
- **Evaluation:** Quarterly reviews + full-time offer decision
